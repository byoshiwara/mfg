Example 2.96.

  1. Determine the domain and range of the function \(h\) graphed at right.
  2. For the indicated points, show the domain values and their corresponding range values in the form of ordered pairs.
Solution.
  1. All the points on the graph have \(v\)-coordinates between \(1\) and \(10\text{,}\) inclusive, so the domain of the function \(h\) is the interval \([1, 10]\text{.}\) The \(h(v)\)-coordinates have values between \(-2\) and \(7\text{,}\) inclusive, so the range of the function is the interval \([-2, 7]\text{.}\)
  2. Recall that the points on the graph of a function have coordinates \((v, h(v))\text{.}\) In other words, the coordinates of each point are made up of a domain value and its corresponding range value. Read the coordinates of the indicated points to obtain the ordered pairs \((1, 3)\text{,}\) \((3,-2)\text{,}\) \((6, -1)\text{,}\) \((7, 0)\text{,}\) and \((10, 7)\text{.}\)
