The range of the linear function \(f (x) = b + mx\) (if \(m \ne 0\)) is also the set of all real numbers, because the graph continues infinitely at both ends. (See FigureÂ 2.6.16a.) If \(m = 0\), then \(f (x) = b\), and the graph of \(f\) is a horizontal line. In this case, the range consists of a single number, \(b\).
